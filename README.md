# EverGreen Box

A project for EE3070 course.

## Overview

This is a web-based application for the EverGreen Box project. It provides an interactive interface for monitoring and controlling the system.

## Project Structure

- `index.html` - Main HTML file
- `script.js` - JavaScript functionality
- `style.css` - Styling
- `timelapse sample/` - Sample timelapse data and resources

## Getting Started

Open `index.html` in your web browser to run the application.

## Features

- Interactive web interface
- Real-time monitoring capabilities
- Responsive design

## Technologies

- HTML5
- CSS3
- JavaScript
